# Background Materials

Place reference materials for your document in this folder.

Supported file types:
- Markdown (*.md)
- PDF (*.pdf)
- Text (*.txt)
- CSV/Excel (*.csv, *.xlsx)
- Code files (*.py, *.r, *.sql, etc.)
- Any other relevant materials

## Example

For a machine learning project specification:
- `requirements.md` - Project requirements
- `data_dictionary.csv` - Dataset schema
- `research_papers/` - Relevant papers
- `sample_code.py` - Reference implementation

The Writer and Reviewer agents will have access to all files in this folder.
